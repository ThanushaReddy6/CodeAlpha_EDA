# 📊 Exploratory Data Analysis — Titanic Dataset
## CodeAlpha Data Analytics Internship | Task 2

## 📌 Project Overview
Performed EDA on the Titanic dataset to find
patterns and insights using Python in Google Colab.

## 🛠️ Tools Used
- Python 3
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Google Colab

## 🔍 Analysis Performed
1. Dataset Overview & Structure
2. Missing Value Detection & Handling
3. Statistical Summary
4. Survival Rate Analysis
5. Age Distribution Analysis
6. Gender vs Survival Analysis
7. Passenger Class Analysis
8. Fare Analysis
9. Correlation Analysis

## 📋 Key Findings
- Only 38.4% of passengers survived
- 74% of females survived vs 19% of males
- 1st class had 63% survival vs 24% in 3rd class
- Average passenger age was 29 years
